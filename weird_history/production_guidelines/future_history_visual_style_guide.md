# Future History: Visual Style Guide

## Core Aesthetic: "Lived-In Future"

The Future History visual identity is **NOT** pristine sci-fi. It's a future that people actually live in — worn, used, slightly dingy around the edges. Think of it as the difference between a tech company's promotional render and the actual building six months after people moved in.

**Reference Films/Shows:**
- Blade Runner 2049 (the mundane parts, not the neon)
- The Expanse (working-class corridors)
- Children of Men (near-future that looks like a tired version of now)
- Her (soft, warm, lived-in tech)
- Black Mirror: "Nosedive" (polished surface, cracks underneath)

---

## Prompt Keyword System

### ALWAYS INCLUDE (The Lived-In Layer)
These keywords force the AI away from sterile sci-fi toward inhabited reality:

```
lived-in environment, well-used surfaces, slightly worn edges,
subtle grime and patina, fingerprints on screens,
scuffed floors, coffee stains on desks,
slightly cluttered environment, realistic wear and tear,
not pristine, not sterile
```

### ALWAYS INCLUDE (The Technical Base)
Same as Weird History — high quality, realistic rendering:

```
8k resolution, cinematic lighting, documentary style realism,
highly detailed, shot on 35mm lens, photorealistic, No painting.
Wide landscape composition.
```

### ALWAYS INCLUDE (Color & Mood)
Future History uses a different color palette than Weird History:

```
muted vibrant colors, warm-cool contrast,
golden hour mixed with fluorescent lighting,
slightly desaturated with pops of corporate color,
realistic color grading like a Denis Villeneuve film
```

### DO NOT USE
```
pristine, sterile, gleaming, brand new, spotless, clean white,
utopian, shiny chrome, neon-soaked, cyberpunk,
Star Trek, generic sci-fi, holographic everything
```

## Setting Guidelines

### Corporate/Medical Spaces
The future's corporations look like today's corporations... but slightly more. Same fluorescent lighting, same motivational posters, same slightly uncomfortable chairs — just with holographic displays instead of LCD screens and biometric scanners instead of key cards.

**Prompt additions for corporate/medical:**
```
corporate environment that has been occupied for years,
slightly outdated holographic displays with dead pixels,
motivational poster peeling at one corner,
hand sanitizer station with a smudged touchscreen,
generic corporate carpet with visible foot traffic patterns
```

### Domestic/Home Spaces
People's homes in the 2040s look like upgraded versions of 2024 homes. IKEA furniture with a tech layer. Cluttered kitchen counters. Kids' drawings on smart-fridges.

**Prompt additions for domestic:**
```
lived-in apartment with personal clutter,
mismatched furniture from different decades,
a smart device with a cracked screen still in use,
plants that need watering, books stacked unevenly,
warm imperfect lighting from mixed sources
```

### Public/Urban Spaces
Cities look like today's cities with incremental upgrades. Not flying cars — just more electric vehicles, more delivery drones with faded paint, more digital signage with the occasional glitched ad.

**Prompt additions for public/urban:**
```
near-future city street that looks like today but slightly evolved,
electric vehicles with road dirt, faded paint on delivery drones,
digital signage with one screen glitched or displaying an error,
sidewalk with patched concrete, weathered bike-share station,
overcast sky with realistic urban haze
```

## Character Guidelines (SAME EXTREMES AS WEIRD HISTORY)

The character casting rule doesn't change — people still need to be visually striking:

- **Hot Route:** `incredibly handsome`, `jaw-droppingly gorgeous`, `supermodel features`
- **Character Actor Route:** `bizarre funny facial features`, `comically ugly`, `exaggerated features`
- **Female characters** in image prompts: always explicitly gorgeous and shapely
- **Clothing:** Modern-plus — recognizable brands and styles evolved 20 years, not jumpsuits

**Key clothing prompt additions:**
```
wearing clothes that look like 2024 fashion evolved 20 years,
slightly futuristic but recognizable casual wear,
not a jumpsuit, not a uniform unless workplace requires it,
layered clothing with subtle tech integration like AR glasses
```

### Video-Specific Character Rules
- **Characters do NOT talk in videos.** Mouths should not move or form words. Characters can react, emote, gesture, nod, look around — but never speak on camera.
- **Ambient sound IS allowed.** Environmental audio (machines humming, traffic, clinking glasses) adds immersion. Just no speech from characters — the voiceover narration carries all dialogue.
- **Prompt wording:** Avoid phrases like "says," "speaks," "tells," "asks" in video prompts. Use "gestures toward," "nods at," "points to," "examines" instead.

## 4. Animation and Assembly Notes

- **Ken Burns on Stills:** All static images use a slow, subtle zoom-in (Ken Burns effect) in post-production. The zoom must be very slow so it doesn't distract from the narration.
- **No Zoom on Video:** Hero videos should NEVER zoom in or use Ken Burns effects. They should play out naturally as generated.
- **Still Image Pacing:** We need more stills in our timelines compared to earlier series. The pacing and timing of these stills must match the voiceover script tightly. Align transitions so the visual on screen always directly corresponds to what the narrator is currently saying.
- **Pacing:** Image duration dynamically fills time between the fixed-duration hero videos to match the voiceover length. Stills should ideally not linger for more than 4-6 seconds before transitioning. If a section of narration is long, break it up with more stills.

## The "Contrast Principle"

The most powerful visual tool for Future History is **contrast between futuristic tech and mundane human life:**

| Scene Element | Futuristic | Mundane |
|:---|:---|:---|
| The technology | Bioprinting chamber, holographic UI | Sitting in a waiting room scrolling a phone |
| The environment | Smart glass, biometric doors | Scuffed linoleum, coffee-ring on the desk |
| The person | AR glasses, BCI headband | Messy hair, tired eyes, wrinkled shirt |
| The mood | The future is here | And it needs a paint job |

**This contrast IS the comedy.** The organ loyalty program is absurd because it exists in a world that looks just like ours — with slightly better screens and slightly worse insurance.

## Prompt Template

Every Future History prompt should follow this structure:

```
[CHARACTER DESCRIPTION] + [NEAR-FUTURE SCENARIO] +
[LIVED-IN ENVIRONMENTAL DETAILS] + [EMOTIONAL/HUMAN MOMENT] +
[TECHNICAL STYLE MODIFIERS]
```

**Example:**
> A ruggedly handsome man in his 40s wearing a slightly wrinkled tailored suit sits in a medical consultation room circa 2048. The room has holographic displays but also a motivational poster peeling at one corner and a hand sanitizer station with a smudged screen. A doctor shows him a holographic heart display. He points at it approvingly. The lighting is warm but slightly harsh from overhead fluorescent mixed with the blue glow of the holo-display. Realistic wear and tear on furniture, scuffed floor. Wide landscape composition. 8k resolution, cinematic lighting, documentary style realism, highly detailed, shot on 35mm lens, photorealistic. No painting.
